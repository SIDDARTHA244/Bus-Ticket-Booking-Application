# Bus-Ticket-Booking-Application

## About
This project simulates a bus-ticket booking system where users can:
- Browse available bus routes
- View bus details and types (AC / Non-AC / Sleeper)
- Choose seats, book tickets, and view their bookings
- Admin side to manage routes, buses, and bookings (if implemented)

## Features
- Search bus routes by source, destination & date
- Show bus options: operator, bus type, timing, fare
- Seat selection interface
- Booking confirmation & user booking history
- User authentication (signup / login)
- Admin interface (manage buses & routes) (if implemented)
- Responsive UI for desktop & mobile

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript,React
- **Backend:** Node.js 
- **Database:** MongoDB 
- **API:** RESTful endpoints for search, bookings, authentication
- **Other:** JWT for auth, bcrypt for passwords, payment gateway stub
- **Deployment/Dev:** npm/yarn, environment variables
